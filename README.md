# Dragon Village 3 攻略站

一個 **純前端、零建置** 的攻略網站。攻略分成三類：**無課・小課 / 中課 / 大課**，
可搜尋、可用標籤篩選，並內建一個「新增 / 管理」編輯器，讓你不用懂程式也能新增攻略。

---

## 1. 檔案結構

```
dv3-guide/
├── index.html          ← 首頁(攻略總覽、分類、搜尋)
├── guide.html          ← 單篇攻略頁
├── editor.html         ← 新增 / 管理攻略(後台編輯器)
├── README.md           ← 這份說明
└── assets/
    ├── css/style.css   ← 樣式(奇幻可愛 × 亮暗模式)
    ├── js/
    │   ├── common.js   ← 共用程式
    │   ├── index.js    ← 首頁邏輯
    │   ├── guide.js    ← 攻略頁邏輯
    │   └── editor.js   ← 編輯器邏輯
    ├── data/guides.js  ← ★ 所有攻略內容都在這裡(唯一要更新的檔)
    └── img/            ← 放你的截圖 / 封面圖
```

> **重點：日常你只會動到兩個東西** —— 用 `editor.html` 編輯、產生新的 `assets/data/guides.js`。

---

## 2. 怎麼新增一篇攻略(最簡單的方式)

1. 打開 **`editor.html`**(本機雙擊或在網站上點「新增/管理」)。
2. 按 **「＋ 新增攻略」**。
3. 填寫：標題、分類(可複選)、標籤、摘要、封面、內文。右邊有 **即時預覽**。
4. 內文用 Markdown，上面有工具列按鈕(標題 / 粗體 / 清單 / 表格 / 圖片 / 影片 / 連結 / 引言)。
5. 編好後按下方 **「⬇ 下載 guides.js」**。
6. 把下載到的 `guides.js` **覆蓋** `assets/data/guides.js`(GitHub 上傳同名檔即可)。
7. 重新整理網站,新攻略就出現了。

> 編輯中的內容會自動暫存在瀏覽器(換頁、關掉再開都還在)。
> 想從頭來、改用目前線上的資料,按「↺ 重新讀取資料檔」。
> 換電腦編輯時,可用「📂 匯入既有 guides.js」把目前的資料載進編輯器繼續改。

---

## 3. 怎麼放圖片

1. 把圖片檔(png / jpg / gif / svg)放到 **`assets/img/`** 資料夾。
2. 在內文用這個格式插入(編輯器「圖片」按鈕會幫你帶出來)：

   ```
   ![圖說文字](assets/img/你的檔名.png)
   ```

3. 封面圖也一樣,在「封面圖」欄位填 `assets/img/你的檔名.png`。
4. 也可以直接貼 **外部圖片網址**(http 開頭)。

---

## 4. 怎麼放影片

把 YouTube 連結 **單獨放一行**,就會自動變成播放器:

```
https://www.youtube.com/watch?v=影片ID
```

(支援 youtube.com、youtu.be、Shorts 連結。)

## 5. 三大分類

| 分類 | 寫進 tiers 的字 | 適合 |
| --- | --- | --- |
| 無課・小課 | `無課小課` | 不花錢或小額也能照做 |
| 中課 | `中課` | 中等課金的配置與取捨 |
| 大課 | `大課` | 追求極致戰力的課金攻略 |

一篇攻略可以同時屬於多個分類(在編輯器勾多個即可)。

## 6. 部署到 GitHub Pages(免費、會有網址)

1. 到 [github.com](https://github.com) 註冊 / 登入,按右上角 **New repository** 建一個新倉庫(例如 `dv3-guide`),設為 **Public**。
2. 進入倉庫 → **Add file → Upload files**,把 `dv3-guide` 資料夾裡的 **所有檔案與資料夾** 拖進去上傳,然後 **Commit**。
3. 倉庫上方 **Settings → Pages**。
4. **Build and deployment → Source** 選 **Deploy from a branch**;Branch 選 **main**、資料夾選 **/ (root)**,按 **Save**。
5. 等 1～2 分鐘,重新整理該頁,上方會出現你的網址(類似 `https://你的帳號.github.io/dv3-guide/`)。把這個網址分享給朋友即可。

之後要更新攻略:在編輯器產生新的 `guides.js` → 到倉庫 `assets/data/` → 上傳同名檔覆蓋 → Commit。約 1 分鐘後網站自動更新。

---

## 7. 本機預覽

直接 **雙擊 `index.html`** 就能用(資料用 `.js` 載入,所以不用架伺服器)。
需要連網才能載入字型與 Markdown 套件;離線時文字仍可讀,只是排版會比較陽春。

---

## 8. 常見問題

- **圖片不顯示?** 確認圖片真的有放進 `assets/img/`,而且路徑、檔名(含大小寫)完全一致。
- **新攻略沒出現?** 確認有用新的 `guides.js` 覆蓋舊檔,並重新整理(必要時清快取)。
- **網址 id 重複?** 每篇 `id` 必須唯一。留空的話,匯出時會自動由標題產生並去除重複。
- **想改網站標題 / 副標?** 在編輯器最下方「站台設定」修改,會一起寫進 `guides.js`。
